# LOOP Contract Spine Nomenclature v0.1

## Locked umbrella name
**LOOP Contract Spine**

Use this as the umbrella name for the contract-layer system that governs safe information movement across source surfaces, Claude.ai chat/cloud, Floor execution, and Node canonical state.

## Locked subnames
- **Ingress Lane** = source intake + normalization
- **Distillation Lane** = reduction into structured meaning
- **Validation Layer** = validation rules and outcomes
- **Adapter Layer** = surface-specific wrappers
- **Compile Boundary** = boundary into Node / canonical state
- **Receipt Layer** = audit trail / mutation records

## Retired umbrella names
These may appear in historical artifacts but should not be used as the active umbrella name:
- YAML Family
- Ingestion Layer
- Distillation

## Usage rule
- Use **Contract Spine** when referring to the system as a whole.
- Use **Ingress Lane** and **Distillation Lane** when referring to individual movement lanes.
- Do not use implementation terms like YAML as the system name.
